What is this?

Potlucky is a tool to help party planners be more organized.

You can create a quick, simple, custom online sign-up sheet for friends and family. As the event host, you pick the categories so whether it's food, drinks, or volunteers, Potlucky.com makes it easy to plan events!

If you have any feedback, please contact us.
